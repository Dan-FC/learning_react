Pasos para iniciar un entorno de React:

1. Para crearlo con vite se escribe en la terminal "npm creat vite@latest" "y" "Project name"

2. Seleccionar React

3. Seleccionar Typescript

4. escribir en la terminal
	cd "Project name"
	npm i
	npm run dev

5. escribor en terminal 
	npm i bootstrap@latest

6 tal vez eliminar index.css pero si vaciar app.css

7. en main.tsx escribir
	import 'bootstrap/dist/css/bootstrap.css'
